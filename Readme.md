PHOTOGRAPHY WEBSITE
This is a simple photography website that has been built using HTML and CSS.

About
The project is a photography website that is designed to showcase visuals that are stunning from different locations. It includes a bio section, a gallery section and a contact section, with simple layout to view easily.
